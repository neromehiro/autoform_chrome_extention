# AutoForm (test autofill)

フォームの入力欄に自動で `test`（または型に応じた定型値）を入れるだけの最小Chrome拡張（テスト用）。

## 使い方
1. このフォルダをローカルに用意（`manifest.json` と `content.js` があること）
2. Chrome で `chrome://extensions` を開く
3. 右上「デベロッパーモード」を ON
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択
5. 何かフォームのあるページを開く（例：会社サイトの問い合わせフォームなど）
6. 入力欄（text/textarea/email/tel/url）が自動で埋まるのを確認

### 注意
- パスワード欄があるページでは **何もしません**（誤入力回避のため）
- 送信はしません／cURL 取得もしません（テスト段階）
- 反映されない場合は、ページをリロードしてみてください
- SPA などでフォームが遅れて表示されるケースでも、対象の入力欄が3つ以上そろったタイミングで自動入力が走ります
- ポップアップ内の「フォーム入力を実行」ボタンを押すと、いつでも強制的に再入力できます（自動入力が済んだ後でも可）
- デバッグ用に、入力欄を検知した件数がページ右上にトースト表示されます

## デバッグ用 JSON インポート
`sample_code/sample_data/apodori_ai.json` のような出力を使って、拡張機能から任意の XPath へ値を投入できます。

1. ブラウザ右上の AutoForm アイコンをクリック
2. 「JSON をインポート」から `.json` ファイルを選択（配列形式であること）
3. 「入力を実行」を押すと、現在アクティブなタブ上で JSON に含まれる XPath/値が順番に適用されます
   - `match_condition` または値に `click` が含まれるものはクリック処理になります
   - input/textarea/select/contentEditable には指定の値を入力し、`input/change` を発火します
4. 状態ラベルで成功/スキップ件数を確認できます

JSON はローカルストレージに保持されるため、次回ポップアップを開いたときも同じ内容で即実行できます。

iframe 内のフォームにも同じ命令が届くよう、拡張機能ボタン経由の JSON 実行／手動入力はタブ内の全フレームへ配送されています。
