# Aimsales AutoForm

**Aimsales AutoForm** は、フォーム営業を自動化できる Chrome 拡張機能です。  
面倒なフォーム入力をなくし、営業メッセージを素早く送れるようにします。  
開発・提供：**Aimsales株式会社**

---

## 概要

フォーム営業や資料請求、問い合わせ対応など、似たような入力を何度も繰り返していませんか？  
Aimsales AutoForm は、そうしたフォーム入力を自動化し、ワンクリックで下書きを挿入できるツールです。

---

## 主な機能

- **ワンクリック入力**：登録しておいた情報をボタンひとつでフォームに自動入力  
- **埋め込みフォーム対応**：HubSpotなど他ドメインのフォームにも対応（初回のみ許可が必要）  
- **フォーム検出**：ページ内のフォーム数・入力欄数を自動で表示  
- **テンプレート保存**：よく使う自己紹介文や提案メッセージを保存・再利用可能  

---

## データの扱いについて

- 「入力を実行」など操作を行ったときだけ、ページHTMLを一時的にAimsalesサーバーに送信し、構造を解析します。  
- Cookieやパスワードなどの機密情報は送信しません。  
- データは自動入力機能のためだけに利用し、第三者への提供や広告利用は行いません。  

---

## 使い方

1. Chrome に拡張をインストール  
2. 営業フォームのあるページを開く  
3. 拡張アイコンをクリック → 「入力を実行」を押す  
4. 登録済みのテンプレート内容がフォームに自動で入ります
